## Development

1. Clone the repo:

  ```
  $ git clone https://github.com/frenchbread/express-boilerplate.git && cd express-boilerplate
  ```

2. Install dependencies:

  ```
  $ npm install && bower install
  ```

3. Run the app:

  ```
  $ npm run dev
  ```

## Tests

```
$ npm test
```

## Build & Run in production

Build
```
$ npm run build
```

Run
```
$ npm start
```
